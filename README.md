## Acerca de Laravel

Laravel es un framework de aplicaciones web con una sintaxis expresiva y elegante. Creemos que el desarrollo debe ser una experiencia agradable y creativa para ser realmente satisfactoria. Laravel facilita el desarrollo al simplificar tareas comunes utilizadas en muchos proyectos web, como:

- Motor de rutas simple y rápido  
- Potente contenedor de inyección de dependencias  
- Múltiples opciones para almacenamiento de sesiones y caché  
- ORM de base de datos expresivo e intuitivo  
- Migraciones de base de datos independientes del motor  
- Procesamiento robusto de tareas en segundo plano  
- Transmisión de eventos en tiempo real  

Laravel es accesible, potente y proporciona las herramientas necesarias para aplicaciones grandes y robustas.

## Aprendiendo Laravel

Laravel cuenta con una de las documentaciones más completas y una gran biblioteca de tutoriales en video, lo que facilita comenzar a usar este framework.

Además, Laracasts ofrece miles de tutoriales en video sobre temas como Laravel, PHP moderno, pruebas unitarias y JavaScript.

También puedes aprender con lecciones prácticas en Laravel Learn, donde crearás proyectos reales desde cero mientras aprendes los fundamentos de PHP.
